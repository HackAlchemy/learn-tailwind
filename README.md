## Run Locally

Clone the project

```bash
  git clone https://github.com/HackAlchemy/learn-tailwind.git
```

Go to the project directory

```bash
  cd learn-tailwind
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
